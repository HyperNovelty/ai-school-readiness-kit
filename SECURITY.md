# Security

This repository is designed for local-only readiness review.

Do not place real student data, school account data, staff records, assessment records, private cases, credentials, API keys, tokens, unpublished materials, or confidential source traces in this repository.

The included scripts use Python standard library only and do not call external APIs. The example data is synthetic. If you adapt the kit for local use, keep private data outside the repository and review any generated files before sharing them.

If you find a safety issue in the kit content or scripts, document it locally and revise the affected file before use. This starter repo does not provide a hosted reporting channel.
