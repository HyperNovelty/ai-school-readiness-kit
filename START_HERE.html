AI School Readiness Kit

This local kit helps teachers, parents, school leaders, tutors, and community learning programs review AI use before it affects assignments, assessment, privacy, access, or student trust.

Start with the checklists

Read docs/field-guide.md, then use the checklist in kits/classroom-ai-use-checklist.md or kits/assignment-review-checklist.md.

Validate and render a readiness card

Run python3 scripts/validate_readiness_card.py examples/school-ai-readiness-card.example.json.

Then run python3 scripts/render_readiness_card_html.py examples/school-ai-readiness-card.example.json examples/rendered/school-ai-readiness-card.example.html.

This kit is not legal, compliance, medical, or mental-health advice. It does not authorize policy decisions, store student data, or call external APIs.
