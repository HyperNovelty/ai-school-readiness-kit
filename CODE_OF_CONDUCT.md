# Code of Conduct

This project is intended for practical public-good work in education contexts.

Participants should use plain, respectful language and focus on reducing confusion, risk, and avoidable harm. Do not add private student information, private school cases, confidential records, or identifying details.

Disagreements should be handled through evidence, local context, and clear boundaries. The kit should remain useful to teachers, families, school leaders, tutors, and community programs without turning into a policy manifesto.
