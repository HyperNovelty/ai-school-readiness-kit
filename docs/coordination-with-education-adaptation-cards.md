# Coordination With Education Adaptation Cards

This kit coordinates with the public `education-adaptation-cards` repository:

https://github.com/HyperNovelty/education-adaptation-cards

The adaptation cards are useful for discussion, scenario framing, and shared vocabulary. This kit has a different job: it helps a local group decide whether a proposed AI use is ready for classroom or learning-program review.

Do not copy private materials into this repository. Do not duplicate the public card deck. Link to the public repository when needed, and keep this kit focused on checklists, local readiness cards, and practical review steps.
